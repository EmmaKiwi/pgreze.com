# pgreze.com

My [hugo](https://gohugo.io/) powered and [netlify](https://netlify.com) hosted blog.

## Usage

```
# Install
brew install hugo

# Start server
hugo server
# With drafts
hogu server -D

# Create release in public/
hugo
```
